# 排程規則

這份文件記錄診所與醫師預約排程的規則決策。

## 核心模型

排程系統分成四種編輯模式：

1. 診所固定模式
2. 診所臨時模式
3. 醫師固定模式
4. 醫師臨時模式

診所可營業時間是上層邊界。醫師上班與預約時間必須在診所開放時間內編輯。

## 診所固定模式

診所固定模式用來設定診所每週固定營業時間。

例如：

```text
星期二 09:00-12:00
星期六 公休
```

這是診所的每週營業基準。

## 診所臨時模式

診所臨時模式用來設定單日例外。

例如：

```text
2026-05-26 臨時公休
2026-05-26 臨時營業 15:00-18:00
2026-05-26 縮短營業 09:00-10:00
```

診所臨時狀態的優先權高於診所固定營業時間。

如果診所某一天公休，該日所有醫師都不可預約，包含醫師固定班表與醫師臨時設定。

## 醫師固定模式

醫師固定模式用來設定某位醫師在某間診所的每週固定上班時間。

醫師固定上班時間必須落在同星期的診所固定營業時間內。

例如：

```text
診所星期二固定營業：09:00-12:00
醫師選擇範圍：10:00-12:00
套用每週二
```

結果：

```text
醫師星期二固定上班時間：10:00-12:00
```

診所時間是可設定邊界。醫師選擇的範圍才是醫師實際固定上班時間。

醫師固定模式用來設定每週固定上班時間：

```text
套用每週二
```

單日或單一窗口異動可使用醫師臨時模式。臨時模式預設處理單日調整，也可以選擇將本次設定同步更新到醫師每週固定規則。

## 醫師臨時模式

醫師臨時模式用來設定醫師單日調整。

醫師臨時模式同樣受該日期的診所有效開放時間限制。

醫師臨時模式預設套用單日，也可以選擇同步更新到每週固定規則。

例如：

```text
醫師今日臨時加開
醫師今日停診
醫師今日縮短上班
將今日設定同步為每週三固定上班
```

## 時間來源規則

醫師固定模式的時間下拉選項來源：

```text
所選星期的診所固定營業時間
```

醫師臨時模式的時間下拉選項來源：

```text
所選日期的診所有效開放時間
```

診所有效開放時間代表：

```text
如果該日期有診所臨時設定，使用診所臨時設定
否則使用該星期的診所固定營業時間
```

## 醫師時間編輯規則

在醫師固定模式與醫師臨時模式中：

1. 時間下拉選項只顯示診所開放時間。
2. 新增、編輯、刪除控制不能產生超出診所開放時間的醫師時段。
3. 刪除醫師時段只影響醫師，不影響診所。
4. 儲存時仍需再次驗證醫師時段是否落在診所開放時間內。

## 預約格產生規則

預約格由醫師上班時間與間隔分鐘數產生。

例如：

```text
醫師上班時間：10:00-12:00
間隔：30 分鐘
產生預約格：10:00、10:30、11:00、11:30
```

整個預約區間必須完整落在醫師上班範圍內。

以 30 分鐘為例：

```text
11:30-12:00 合法
12:00-12:30 不合法
```

## 暫停預約格

醫師上班時間與預約格是不同概念。

例如：

```text
醫師上班時間：10:00-12:00
暫停預約格：10:30
```

醫師上班時間仍然是：

```text
10:00-12:00
```

可預約格是：

```text
10:00、11:00、11:30
```

## 合法預約時間

一個時間可以被病患預約，必須同時符合：

1. 診所該日沒有公休或臨時停業。
2. 醫師該日沒有停診。
3. 預約區間落在醫師上班時間內，或是合法的醫師單日調整內。
4. 該預約格沒有被醫師暫停。
5. 該預約格沒有被其他病患預約。
6. 該預約格不是過去時間。
7. 整個預約區間完整落在允許範圍內。

## 優先權順序

解析可預約狀態時使用以下優先順序：

```text
診所臨時公休
> 診所臨時營業 / 縮短營業
> 醫師今日停診
> 醫師今日臨時調整
> 醫師每週固定上班時間
> 診所每週固定營業時間
```

簡化理解：

```text
診所關門，所有醫師都停診。
診所有開，醫師才可以在診所開放時間內排班。
醫師固定模式用來套用每週固定規則。
醫師臨時模式可處理單日或單一窗口異動，也可選擇把本次設定同步更新到每週固定規則。
```

## UI 文案

建議模式文案：

```text
診所固定模式：固定模式 / 固定套用每週
診所臨時模式：臨時模式 / 只套用今天
醫師固定模式：固定模式 / 固定套用每週
醫師臨時模式：臨時模式 / 只套用今天 / 更新每週固定
```

醫師頁應使用醫師語意：

```text
醫師上班時間
醫師今日停診
套用每週二
只套用今天
```

診所頁應使用診所語意：

```text
診所營業時間
今日公休
臨時停業
```
